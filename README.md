# Retail & Warehouse Sales Analysis Dashboard

## Project Overview
This project analyzes retail and warehouse sales data using Microsoft Excel to identify trends in supplier performance, product demand, and sales channels.

## Dataset
The dataset contains 30,000 sales records including supplier information, product categories, and sales values.

## Tools Used
- Excel
- Power Query
- Pivot Tables
- Data Visualization

## Dashboard Features
- Sales KPI indicators
- Monthly sales trends
- Supplier performance analysis
- Product category insights
- Retail vs warehouse sales comparison

## Key Insights
- Top suppliers contribute majority of revenue
- Certain product categories dominate sales
- Warehouse sales play a significant role in revenue
